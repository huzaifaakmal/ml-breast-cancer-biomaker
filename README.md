# A Compact 5-Gene Signature for Breast Cancer: Computational Pipeline & Figures

## Authors
- **Researcher:** Huzaifa Akmal (Sanata Dharma University, Yogyakarta, Indonesia)
- **Supervisor:** Narendar Kumar (University of Technology of Belfort-Montbéliard (UTBM), Belfort, France)

This repository provides the computational scripts and research figures for the study: **"A Compact 5-Gene Signature (FHL1, PCK1, ACVR1C, FABP4, and MT1H) for Diagnostic Prediction and Immune Characterization in Breast Cancer: An Integrated Machine Learning Approach."**

The project focuses on identifying a robust molecular signature using an ensemble machine learning approach and characterizing its relationship with the tumor immune microenvironment.

---

## 📂 Project Structure

### 1. `Scripts/`
This directory contains the Python-based analytical pipeline, organized sequentially:

- **`01_Master_Pipeline.py`**: The primary entry point for the analysis, handling data ingestion and initial processing.
- **`02_Master_Pipeline_Expanded.py`**: An extended version of the pipeline for broader cohort analysis and robustness checks.
- **`03_Model_Pipeline.py`**: Implements the core Machine Learning models (LASSO and SVM-RFE) for feature selection.
- **`04_DEA_Enrichment.py`**: Performs Differential Expression Analysis (DEA) and Functional Enrichment (GO/KEGG).
- **`05_Performance_Interpretability.py`**: Calculates model performance metrics (AUC, MCC) and generates SHAP interpretability values.
- **`06_Immune_Validation.py`**: Integrates CIBERSORT results to validate the signature against the immune microenvironment.
- **`07_Survival_Batch.py`**: Handles survival analysis and batch effect assessment across different datasets.
- **`08_Implement_Final_Figure_Plan.py`**: The master script for generating all publication-quality figures found in the `Figure/` folder.
- **`Data/`**: (Sub-directory) Placeholder or location for the transcriptomic datasets (GSE42568, GSE15852, etc.).

## 📄 Abstract
Breast cancer remains a significant clinical challenge due to its molecular heterogeneity. This study identifies a robust **5-gene signature (FHL1, PCK1, ACVR1C, FABP4, MT1H)** that targets metabolic and growth hubs. Using an ensemble machine learning approach (LASSO + SVM-RFE) on transcriptomic data from multiple cohorts (GSE42568, GSE15852, TCGA-BRCA, and METABRIC), we developed a parsimonious molecular tool with high diagnostic accuracy (Discovery AUC: 0.91, Validation AUC: 0.89).
